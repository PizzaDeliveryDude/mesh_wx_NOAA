# mesh_wx_NOAA
Uses NOAA and Navy APIs to send over Meshtastic. Still a variation in progress, but making headway

# Needs jq installed

# Usage for noaa_full.sh

./noaa_full.sh KLGA "Citi Field"

## First variable is airport code. Defaults to KNYC

## Second variable is sent from location. Defaults to airport code

# Usage for navy_sun_moon.sh

./navy_sun_moon.sh KLGA "Citi Field"

## First variable is airport code. Defaults to KNYC

## Second variable is sent from location. Defaults to airport code

# Please make sure to use both variables to ensure your message is accurate
